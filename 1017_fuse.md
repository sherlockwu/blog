# FUSE 使用

## 核心:
* 安装 FUSE (user space lib, 内核中应该已经有了)
* 挂载新file system 到 FUSE的 dir之下
* FUSE 就会帮我们把user 向kernel的system call 全部redirect 到我们的实现，然后在通过kernel 返回给user
