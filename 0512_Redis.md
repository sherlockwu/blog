# Redis & MongoDB 
写这篇Blog的目的是学习一下Redis（它据称是NoSQL的DB）

## 整体的感觉
应该都是NoSQL的DB，但是Redis完全在Mem中，所以可能速度更快；而MongoDB亮点在于提供复杂数据结构存储的支持
但是，还是先把SQL学完，然后，开始看一些这个MongoDB，然后Redis

## 学习
可以先看Redis入门，然后再学一学怎么用
